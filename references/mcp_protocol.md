# HUSKYLENS 2 MCP 协议技术细节

## SSE连接

HUSKYLENS 2 MCP服务通过SSE (Server-Sent Events) 协议通信。

**端点格式：** `http://<device-ip>:3000/sse`

**连接流程：**
1. 客户端向SSE端点发起HTTP GET请求
2. 服务端返回SSE事件流
3. 客户端解析事件流获取工具列表和调用结果

## 工具详细说明

### manage_applications

管理HUSKYLENS 2的AI算法/模型。

**功能：**
- 列出所有可用的AI算法
- 切换当前运行的算法
- 查询当前运行的算法名称

**支持的算法（25种）：**

| 序号 | 算法名称 | 功能说明 |
|-----|---------|---------|
| 1 | 人脸检测 | 检测画面中的人脸位置 |
| 2 | 人脸识别 | 检测并识别已学习的人脸 |
| 3 | 人脸五官检测 | 检测人脸五官关键点 |
| 4 | 物体识别 | 识别80种COCO类别物体 |
| 5 | 物体追踪 | 追踪指定物体运动 |
| 6 | 颜色识别 | 识别和区分颜色 |
| 7 | 物体分类 | 图像分类 |
| 8 | 自学习分类 | 用户自定义分类训练 |
| 9 | 实例分割 | 像素级目标分割 |
| 10 | 手掌检测 | 检测手掌位置 |
| 11 | 手掌关键点检测 | 检测手掌21个关键点 |
| 12 | 手势识别 | 识别手势动作 |
| 13 | 人体检测 | 检测人体位置 |
| 14 | 人体关键点检测 | 检测人体骨骼关键点 |
| 15 | 姿态识别 | 识别人体姿态 |
| 16 | 车牌识别 | 识别车牌号码 |
| 17 | 文字识别 | OCR文字识别 |
| 18 | 巡线追踪 | 巡线导航 |
| 19 | 表情识别 | 识别面部表情 |
| 20 | 注视方向检测 | 检测眼睛注视方向 |
| 21 | 人脸朝向检测 | 检测人脸朝向 |
| 22 | 标签识别 | 识别AprilTag等标签 |
| 23 | 二维码识别 | 识别QR码 |
| 24 | 条形码识别 | 识别条形码 |
| 25 | 跌倒检测 | 检测人体跌倒 |

**模糊匹配说明：** 算法切换支持模糊识别，例如指令"人脸检测"会自动匹配到"人脸识别"。

### multimedia_control

控制HUSKYLENS 2的多媒体功能。

**功能：**
- 拍照：拍摄当前摄像头画面，照片保存到U盘系统

### get_recognition_result

获取当前算法的识别结果。

**返回信息：**
- 检测到的目标列表
- 每个目标的ID、名称、置信度
- 目标在画面中的位置坐标（中心点x,y和宽高w,h）

### task_scheduler

设置条件触发的自动任务。

**功能：**
- 当指定目标出现在画面时执行操作（如拍照）
- 触发机制：目标出现后执行一次，目标消失后重新出现才会再次执行
- 拍摄的照片存储在U盘系统的photo文件夹中

## 终端调试

SSH连接HUSKYLENS 2：
```
ssh root@192.168.88.1
```

**配置文件：**
- `llm_agent.json` — 大模型API配置（api_key等）
- `huskylens_mcp_server.json` — MCP服务工具描述和提示词

**手动启动MCP服务：**
```bash
/opt/huskylens_mcp_server
```

**检查MCP服务状态：**
```bash
ps -ef | grep mcp
```

**运行内部智能体：**
```bash
./llm_agent
```
内部智能体可获取摄像头画面，对"你看到了什么"有更准确的回答。
